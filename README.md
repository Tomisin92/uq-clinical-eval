# Uncertainty Quantification in Clinical ML — Full Pipeline

## Project Structure

```
uq_paper/
├── data/
│   ├── raw/                  ← Put MIMIC-III CSVs here
│   ├── processed/            ← Auto-generated by preprocess.py
│   └── preprocess.py         ← Step 1: extract features
├── models/
│   ├── models.py             ← BaseNet, MCDropout, BayesNet (BNN)
│   ├── train.py              ← Training loops
│   └── conformal.py          ← Split conformal prediction
├── evaluation/
│   └── evaluation.py         ← ECE, ACE, Brier, net benefit, stats tests
├── results/                  ← Auto-generated by run_experiments.py
├── run_experiments.py        ← Step 2: run all methods, save JSON results
├── generate_latex_tables.py  ← Step 3: convert results to LaTeX tables
├── paper.tex                 ← Full LaTeX paper (paste real tables in)
└── requirements.txt
```

## Quickstart

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Get MIMIC-III data
- Register at https://physionet.org (requires CITI training)
- Download MIMIC-III CSVs → place in `data/raw/`
- Required files: ADMISSIONS.csv, PATIENTS.csv, ICUSTAYS.csv,
  CHARTEVENTS.csv, LABEVENTS.csv

### 3. Preprocess
```bash
cd data
python preprocess.py
```
Outputs: `data/processed/mortality_features.csv`, `readmission_30d_features.csv`

### 4. Run experiments (GPU recommended, ~2–4 hours on CPU)
```bash
python run_experiments.py
```
Outputs: `results/results_raw.json`, `results/results_summary.json`,
`results/stats_ece.json`, etc.

### 5. Generate LaTeX tables
```bash
python generate_latex_tables.py
```
Outputs: `results/latex_tables.tex`

### 6. Compile paper
In `paper.tex`, replace the three placeholder tables with:
```latex
\input{results/latex_tables}
```
Then compile:
```bash
pdflatex paper.tex
pdflatex paper.tex   # run twice for references
```

## Key Design Decisions

- **Imputer/scaler fit on training data only** — no data leakage
- **Conformal calibration set carved from training period** — no test contamination
- **Time-based split** — 2001–2008 train, 2009–2012 shifted test
- **5 seeds** — mean ± SD reported for all metrics
- **Holm-Bonferroni correction** — controls family-wise error rate across 3 pairwise comparisons
- **BNN uses local reparameterisation** — lower gradient variance than standard reparameterisation
